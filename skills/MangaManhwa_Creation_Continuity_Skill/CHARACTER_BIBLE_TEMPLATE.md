## 8. Prompt lock resumido

Escreva uma versão curta para colar em todo prompt:

```text
[Nome], [idade aparente], [altura/corpo], [rosto], [olhos], [cabelo], [roupa atual], [acessórios], [arma/objeto], [ferimentos atuais], [postura], [paleta]. Must remain the same character across every panel/page.
```

## 9. Checklist de aprovação

- [ ] A ficha permite reconhecer o personagem sem nome?
- [ ] Há pelo menos 3 âncoras visuais fortes?
- [ ] Roupa atual está clara?
- [ ] Acessórios/arma têm posição definida?
- [ ] Ferimentos e sujeira têm estado atual?
- [ ] Forma de falar está definida?
- [ ] Existe prompt lock colável?
